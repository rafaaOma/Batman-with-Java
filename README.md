# Batman 2D Graphics (Java)

A simple Java Swing project that draws a Batman-inspired figure using Java Graphics2D.

## Concepts Used
- Java Swing
- Graphics2D
- Shapes and Arcs
- Cubic Bezier Curves
- AffineTransform (rotate, translate)
- Anti-aliasing

## Purpose
This project was created for practicing Java 2D graphics and understanding transformations and curves.
